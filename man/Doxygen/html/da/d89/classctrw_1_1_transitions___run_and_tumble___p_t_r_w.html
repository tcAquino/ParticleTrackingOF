Detailed Description

template<typename Boundary, typename StateSwitcher, typename JumpGenerator, typename OrientationGenerator, typename OrientationGenerator_Wall>
class ctrw::Transitions_RunAndTumble_PTRW< Boundary, StateSwitcher, JumpGenerator, OrientationGenerator, OrientationGenerator_Wall >

Update state according to current internal state.

When internal state is:

  • 0 (run): Update position as according to JumpGenerator object.
    Apply Boundary condition.
    If internal state has not been switched to wall-tumble, apply the StateSwitcher run rule given state to obtain new internal state.
  • 1 (tumble): Apply the StateSwitcher tumble rule given state to obtain new internal state.
    If internal state has been switched to run, update orientation accoiding to OrientationGenerator .
  • 2 (wall-tumble): Apply StateSwitcher wall_tumble rule to obtain new internal state.
    If internal state has been switched to run, update orientation accoiding to OrientationGenerator_Wall .
    Template Parameters
    BoundaryObject to enforce boundary condition on new state given new state and old state.
    StateSwitcherObject to return new internal state given state.
    JumpGeneratorObject to return jump given state.
    OrientationGeneratorObject to return orientation increments given state.
    OrientationGenerator_WallObject to return orientation increments given state.
    Note
    State must define:
  • position
  • orientation
  • state StateSwitcher must define:
  • int run(State const&)
  • int tumble(State const&)
  • int wall_tumble(State const&)

Boundary object should set wall_tumble state when appropriate.

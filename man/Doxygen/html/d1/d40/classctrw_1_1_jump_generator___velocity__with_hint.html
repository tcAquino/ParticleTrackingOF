Detailed Description

template<typename VelocityField>
class ctrw::JumpGenerator_Velocity_withHint< VelocityField >

Jump according to a velocity field and time step (forward Euler) with the previous position as a hint to locate the current position for velocity interpolation.

State must define:

  • position
  • tag (to identify particles for hints)
    Note: Particle number and tags must remain fixed.

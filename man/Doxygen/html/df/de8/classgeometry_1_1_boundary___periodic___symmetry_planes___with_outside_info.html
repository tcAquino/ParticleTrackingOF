Detailed Description

template<typename SymmetryPlanes>
class geometry::Boundary_Periodic_SymmetryPlanes_WithOutsideInfo< SymmetryPlanes >

Periodic boundaries along each symmetry plane with information about where position would be outside domain.

State must define:

  • position (vector type)
  • periodicity (std::vector<int>) counting how many domains have been traveled along each dimension

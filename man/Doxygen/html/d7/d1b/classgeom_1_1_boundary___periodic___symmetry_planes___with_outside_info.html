Detailed Description

template<typename SymmetryPlanes>
class geom::Boundary_Periodic_SymmetryPlanes_WithOutsideInfo< SymmetryPlanes >

Periodic boundaries along each symmetry plane with information about where position would be outside domain.

Note
State must define:
  • position [container]
  • periodicity [container of signed integers] (counting how many domains have been traveled along each dimension)

Detailed Description

template<typename VelocityField, typename Boundary = geometry::Boundary_DoNothing>
class ctrw::JumpGenerator_Velocity_State_RK4< VelocityField, Boundary >

Jump according to a velocity field and time step, using RK4 scheme.

Boundary conditions are enforced in predictor-corrector steps.
State must define:

  • position

Detailed Description

Get projection of absolute position along given direction, given state from position and periodicity.

state.periodicity should hold the (directional) integer number of times unit cell has been traversed.

Note
  • Unit cell faces must be perpendicular to (cartesian) coordinate axes.
  • Direction vector is normalized upon construction.

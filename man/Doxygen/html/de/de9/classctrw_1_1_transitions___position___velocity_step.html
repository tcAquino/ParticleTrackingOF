Detailed Description

template<typename JumpGenerator, typename VelocityField, typename Boundary = geometry::Boundary_DoNothing>
class ctrw::Transitions_Position_VelocityStep< JumpGenerator, VelocityField, Boundary >

Update position according to velocity field with adapative time step to fix jump increments.

Tiime step is determined according to jump sizes divided by local velocity. Jump increments use forward Euler.

Template Parameters
JumpGeneratorObject to return jump given state.
VelocityFieldObject to return velocity value given state.
BoundaryObject to enforce boundary condition on new state given new state and old state.
Note
State must define:
  • position
  • time

Velocity field may be passed to constructor or velocities may be obtained from jump_generator if JumpGenerator defines:

  • velocity(State const&)

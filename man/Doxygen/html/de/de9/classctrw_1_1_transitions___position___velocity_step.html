Detailed Description

template<typename JumpGenerator, typename Velocity, typename Boundary = geometry::Boundary_DoNothing>
class ctrw::Transitions_Position_VelocityStep< JumpGenerator, Velocity, Boundary >

Update state.position according to a JumpGenerator object given state with time step according to prescribed jump size divided by velocity calculated accorsing to Velocity object given state.

Apply Boundary condition given new state and old state.

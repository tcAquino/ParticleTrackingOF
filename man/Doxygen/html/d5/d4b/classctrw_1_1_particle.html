Detailed Description

template<typename State_t>
class ctrw::Particle< State_t >

Particle class for ptrw/ctrw/tdrw methods.

A particle has an old state and a new state.
Transitions are handled by an external class that modifies the state.
Upon a transition, the old state becomes the previous new state and the new state is obtained by applying the transition to the previous new state.

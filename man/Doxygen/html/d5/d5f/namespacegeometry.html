Add periodic velocity point images outside domain.

If the projection of a point is within a given fraction of the boundary along each symmetry plane, add an image along that plane.

Parameters
pointsPoint positions, to add new points.
velocitiesVelocity values, to add new velocities.
fractionFraction of original points to add.
boundary_periodicObject to enforce periodic boundary conditions.
Note

  • symmetry_planes
  • scale
  • origin

inlineprivate

Find average and variance of patch face centers.

Choose dimension with least variance.

Check which dimensions have periodic BCs If a dimension has periodic BCs they must be exactly two.

Periodic conditions are assumed to be checked in order across Cartesian dimensions. If there are periodic BCs in all dimensions starting from 0 up to some dimension, add only those boundaries If there are gap dimensions between periodic dimensions, add {-inf, inf} boundaries.

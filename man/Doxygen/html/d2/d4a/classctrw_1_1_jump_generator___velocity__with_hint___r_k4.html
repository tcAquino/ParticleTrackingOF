Detailed Description

template<typename VelocityField, typename Boundary = geometry::Boundary_DoNothing>
class ctrw::JumpGenerator_Velocity_withHint_RK4< VelocityField, Boundary >

Jump according to a velocity field and time step, using RK4 scheme, with the previous position as a hint to locate the current position for velocity interpolation.

State must define:

  • position
  • tag (to identify particles for hints)
    Note: Particle number and tags must remain fixed.

Detailed Description

template<typename FlowField, typename Boundary = geometry::Boundary_DoNothing, typename RNG = std::mt19937>
class ctrw::Transitions_PTRW_FlowField_Diff< FlowField, Boundary, RNG >

Update position according to prescribed flow field and time step.

Jump increments use forward Euler.

Template Parameters
FlowFieldObject to return velocity value given position.
BoundaryObject to enforce boundary condition on new state given new state and old state.
Note
State must define:
  • position
  • time

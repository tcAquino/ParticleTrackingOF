Detailed Description

template<typename Subject, typename VelocityField, typename Geometry>
struct ptof::Output_Cases< Subject, VelocityField, Geometry >::Output_velocity_gradient

Output time and velocity gradient components.

"PTOF/Output.h"

Component output order is xx, xy, xz, yx, yy, yz, zx, zy, zz. Spurious components are omitted if geometry dimension is < 3.

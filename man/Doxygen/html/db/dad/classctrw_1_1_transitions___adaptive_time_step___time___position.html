Detailed Description

template<typename TimeStepAdaptor, typename TimeGenerator, typename JumpGenerator, typename Boundary = geometry::Boundary_DoNothing>
class ctrw::Transitions_AdaptiveTimeStep_Time_Position< TimeStepAdaptor, TimeGenerator, JumpGenerator, Boundary >

Update position and time according to specified jumps and time increments, using state-based adaptive time steps.

Template Parameters
TimeStepAdaptorObject to return and update time step given state, time generator, and jump generator.
TimeGeneratorObject to return time increment given state.
JumpGeneratorObject to return jump given state.
BoundaryObject to enforce boundary condition on new state given new state and old state.
Note
State must define:
  • position
  • time

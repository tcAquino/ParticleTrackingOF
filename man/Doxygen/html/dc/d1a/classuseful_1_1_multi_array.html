Detailed Description

template<typename value_t>
class useful::MultiArray< value_t >

Array container with arbitrary rank.

Different dimensions may have different sizes. The elements are indexed, traversing the container in the order (0,...,0), (0,...,1), ..., (0,..,1,0) in either 1D or using a multi-dimensional index. The elements are continguous in memory in the order of the 1D index.

Detailed Description

template<typename Transitions_Transport, typename Reaction>
class ctrw::Transitions_PTRW_Transport_Reaction< Transitions_Transport, Reaction >

Update state according to a transport transition given state followed by a reaction transition given state and time step, with a prescribed time step.

Note: transport rule should enforce any boundary conditions.

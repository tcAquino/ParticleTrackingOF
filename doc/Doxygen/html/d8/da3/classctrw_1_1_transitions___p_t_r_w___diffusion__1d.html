Detailed Description

template<typename Boundary = geom::Boundary_DoNothing, typename ParallelOption = par::ParallelOptions::Serial, typename RNG = stochastic::RNGThreaded<ParallelOption, std::mt19937>>
class ctrw::Transitions_PTRW_Diffusion_1d< Boundary, ParallelOption, RNG >

Update position according to diffusion in one dimension with prescribed time step.

Jump increments use stochastic forward Euler.

Template Parameters
BoundaryObject to enforce boundary condition on new state given new state and old state.
ParallelOptionChoose serial or parallel implementation.
RNGRandom number generator.
Note
State must define:
  • position
  • time

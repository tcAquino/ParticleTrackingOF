Detailed Description

template<typename TimeGenerator, typename JumpGenerator, typename Reaction>
class ctrw::Transitions_Reaction_Position< TimeGenerator, JumpGenerator, Reaction >

Update state according to reaction, followed by a position update, followed by a time update.

Template Parameters
TimeGeneratorObject to return time increment given state.
JumpGeneratorObject to return jump given state.
ReactionUpdate particle state according to reaction, given state and time increment.
Note
State must define:
  • position
  • time

JumpGenerator should enforce any boundary conditions.

std::vector<std::size_t> modular::convert ( Integer_Type  to_convert,
Integer_Type  basis,
std::size_t  nr_digits 
)
Returns
Base basis representation of base-10 integer to_convert as a std::ring of digits.

The total number of digits is nr_digits, which must be at least the number of digits needed to represent the number. If bigger, the result is padded with zeros on the left.

Note
Units digit in to_convert[0].

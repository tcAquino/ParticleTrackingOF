Detailed Description

template<typename ParticleMaker, typename Geometry>
class ptof::InitialCondition_PrescribedPositionsMassesTags< ParticleMaker, Geometry >

Initial condition at prescribed positions, masses, and tags, read from a file.

"CTRW/InitialCondition.h"

File should contain particle positions, masses, and tags, one particle per line.

Discrete conditional distribution over given set of values.

"Stochastic/Random.h"

Template Parameters
Discrete_distDistribution conditioned on a given value. Can be a cdf or pdf implementing bin_probability(std::size_t, std::size_t) and value(Scalar, std::size_t), where the second argument refers to the value being conditioned on.

Detailed Description

template<typename Transitions_Transport, typename Reaction>
class ctrw::Transitions_PTRW_Transport_Reaction< Transitions_Transport, Reaction >

Update state according to specified transport transitions followed by reaction.

"CTRW/Transitions.h"

Template Parameters
Transitions_TransportUpdate particle state according to transport.
ReactionUpdate particle state according to reaction, given state and time increment.
Note
  • transitions_transport should enforce any boundary conditions.
  • Time step needed for reaction may be specified or extracted from transport transitions if Transitions_Transport defines:
  1. time_step()

◆ bin_widths() [1/2]

template<typename Container = std::vector<double>>
auto op::bin_widths ( Container const &  values)

Compute widths of bins having given values as midpoints.

  • Leftmost bin edge is the minimum value minus half the width of the first bin.
  • Widths are computed sequentially as twice the distance between the next midpoint and the current left edge.
